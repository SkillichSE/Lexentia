## Skills (инструкции для ассистента)

Эта папка содержит короткие “skill”-файлы, которые ассистент может читать по запросу, чтобы действовать корректно в вашем окружении.

### Как использовать

- Ассистент может предложить действие вида “прочитать skill `<name>`”.
- После чтения он предложит действия (обычно команды терминала), которые **требуют подтверждения пользователя кнопкой**.

### Правила

- Skill-файлы должны быть в корне `skills/` и иметь расширение `.md` или `.txt`.
- В имени файла нельзя использовать поддиректории.
