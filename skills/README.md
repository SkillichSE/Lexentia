## Skills (AI Assistant Instructions)

This folder contains short "skill" files that the AI assistant can read on request to act correctly in your environment.

### How to Use

- The assistant may suggest an action like "read skill `<name>`".
- After reading, it will propose actions (usually terminal commands) that **require user confirmation via button click**.

### Rules

- Skill files must be in the `skills/` root directory and have `.md` or `.txt` extension.
- Subdirectories are not allowed in file names.
