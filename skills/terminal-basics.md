## Терминал: базовые правила выполнения команд

### Контекст

Встроенный терминал работает как обычный shell. Команды могут выполняться **только после подтверждения пользователя**.

### Рекомендации

- Для потенциально опасных команд сначала выводи “dry run” / проверку.
- Для команд, создающих файлы/папки, сначала делай `ls`/проверку.
- Если путь содержит пробелы — используй двойные кавычки:

```bash
cd "/path/with spaces"
```

### Полезные команды

Текущая папка:

```bash
pwd
```

Список файлов:

```bash
ls -la
```

Поиск текста (ripgrep):

```bash
rg "pattern"
```
